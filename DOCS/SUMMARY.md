# Nate's Space - Project Summary

## What Is This?
A personal portfolio/social-style website for Nate, featuring studio content, music updates, and a MySpace-inspired feed layout.

## Tech Stack
- **Frontend**: Pure HTML5, CSS3, JavaScript (no frameworks)
- **Fonts**: Google Fonts (Outfit, Space Mono)
- **Hosting**: GitHub Pages (static)
- **Build**: None required - just push and deploy
